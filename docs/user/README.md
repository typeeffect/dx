# DX User Documentation

This folder documents the use of DX. For design, roadmap, and internals see `docs/`.

## Guides

- [Quickstart](QUICKSTART.md) — from zero to a running native binary
- [Language Guide](LANGUAGE_GUIDE.md) — syntax, types, and core features
- [Effects Guide](EFFECTS_GUIDE.md) — effects, handlers, and why there is no `async`
- [Schema Guide](SCHEMA_GUIDE.md) — typed data access via compile-time schema providers
- [Python Bridge](PYTHON_BRIDGE.md) — interop with the Python ecosystem

## Status Convention

Examples in these guides are marked:

- **works today** — compiles and runs with the current toolchain
- **preview syntax** — intended direction, not yet parser-stable or fully implemented
